Get and install Contenticious

Get the latest Contenticious version, either by downloading and extracting the latest release tarball from the Contenticious tags page or by cloning the whole repository via

$ git clone git://github.com/memowe/contenticious.git

Maybe you'll need to install some dependencies, that are not core modules. Don't worry, not that much. With cpanminus you can install all needed dependencies and Contenticious itself in a very easy way:

$ cd contenticious
$ cpanm --sudo .

Done.

The traditional way is also possible. You can use your standard procedure to install Perl modules since Contenticious ships with a Makefile.PL and tests:

$ perl Makefile.PL
$ make
$ make test
$ sudo make install
