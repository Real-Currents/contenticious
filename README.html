NAME

Contenticious -- simple file based "CMS" on Mojo steroids!

SYNOPSIS

    $ vim pages/index.md
    $ mkdir pages/section
    $ vim pages/section/foo.md
    $ vim pages/section/bar.md
    $ perl contenticious.pl

DESCRIPTION

Contenticious is a very simple way to glue together some content to a small website. You just write Markdown files and check the generated HTML in your browser. To publish, just use the dump command to generate static HTML as described below.
